Tangram is a Web Based lightweight XML Script engine for Desktop Application. It has fast and native support for various component: ActiveX, DotNet Control, MFC View, MFC Window, and Html Page. In Tangram, a general Window Object can host a Web Page without using a web browser control. But you can use it just as you have a web browser control embedding in this window object. For example, you can write JS Code in this page, respond various application event in this page, you can write Application UI Script as DOM does, we named such kind of web page as “Window behind Web Page”.
